info sobre la instalación en una unidad de pc basada en web server IIS v5.0 y bajo 
la plataforma windows xp profesional sp3.


1.- Se requiere S.O Windows XP Profesional SP3 + IIS v5.0. (No se ha testeado y 
    adatado para en versiones superiores en S.O y Web Server IIS).


3.- Copiar todo el contenido del CD en la carpeta 'gpc' dentro de la ruta 
    C:\Inetpub\wwwroot\gpc

3.- Establecer en orígenes de datos (ODBC) el acceso a la base de datos en la 
     carpeta 'data' y la base de datos basada en Access 2003 denominada 'dbgpc.mdb'.

4.- Verificar los privilegios de administrador para el acceso total si limitación a la 
    ejecución y carga de archivos requerido para tal fin. png, jpg, etc... 

5.- Crear un acceso directo al navegador FireFox de Mozilla ejecutando el archivo 'index.asp'
